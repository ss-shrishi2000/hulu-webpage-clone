# hulu-webpage-clone
A clone of Hulu Webpage built with HTML, CSS, and Javascript.
Essential Segments of the project include :-
 1. Base Styling
 2. Media Queries For Responsiveness
 3. Categories And Covers Section
 4. Header Shadows/ Gradient
 5. Modal HTML & CSS
 6. Modal Javascript
